Part 2 — Edit the code for your business

Students take the AI-generated code and make 3 targeted edits to connect it to their own Bito account — without leaking any data in the process.

The 3 things students need to change

① Add your API key (in .env only)

  • Never put the key directly in the code
  • Create a .env file on your machine
  • Add: BITO_API_KEY=your_key
  • .env files stay local — never uploaded or shared

② Choose which data to sync

  • Open sync_runner.py
  • Set True/False next to each data type
  • Only enable what your dashboard needs
  • Less data = faster sync + better privacy

③ Check what fields you export

  • Open bito_client.py
  • Find the function for your data type
  • Remove any fields you don't need
  • Personal fields (phone, address) are already excluded

✓ What never changes

  • The API endpoints stay the same for everyone
  • The CSV output format stays the same
  • Power BI connection stays the same
  • Only credentials + field selection are personal
