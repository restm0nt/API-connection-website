Lesson 1

Setup your computer

Install these tools once. After this you are ready for any template.

1

Install Python

Go to python.org and download the latest version. During install check the box that says 'Add Python to PATH'.

python.org/downloads ↗

After install: open Command Prompt and type python --version. You should see a version number.

2

Install VSCode

Download and install Visual Studio Code. This is your code editor.

code.visualstudio.com ↗
3

Install the Python extension in VSCode

Open VSCode. Click the Extensions icon on the left sidebar (4 squares). Search for 'Python' and install the one by Microsoft.

4

Install required libraries

Open Command Prompt (search for 'cmd' in Start menu) and run this command:

pip install requests python-dotenv

Wait for it to finish. You will see 'Successfully installed' at the end.

1

Install Python

Go to python.org and download the latest version for macOS.

python.org/downloads ↗

After install: open Terminal and type python3 --version.

2

Install VSCode

Download and install Visual Studio Code for Mac.

code.visualstudio.com ↗
3

Install the Python extension in VSCode

Open VSCode. Click the Extensions icon on the left sidebar. Search 'Python' and install the Microsoft one.

4

Install required libraries

Open Terminal (Cmd+Space → Terminal) and run:

pip3 install requests python-dotenv
All done? Go to Lesson 2 to get your API key, download your template and connect your data. → Lesson 2
Lesson 2

Download & run your template

First get your Bito API key, then pick a template and follow 3 steps.

0

Get your Bito API key

Log in to Bito. Go to Settings → Integrations. Copy the API key. Keep it safe — treat it like a password.

Never share your API key — not in chat, not in email, not in code you send to someone else.

After downloading — 3 things to edit

① Add your API key

Open the .env file inside the folder. Paste your Bito API key after the = sign. Save. That's it.

BITO_API_KEY=paste_your_key_here

② Remove columns you don't need

Open sync_runner.py in VSCode. Find the section marked EDIT THIS. Remove any column name you don't want in your CSV. Remove the matching value on the next line too.

③ Run the script

Open the folder in VSCode (File → Open Folder). Open the terminal (Terminal → New Terminal). Run the command below. Your CSV file will appear in the same folder.

python sync_runner.py

On Mac use python3 instead of python.

Connect to Power BI

1

Open Power BI Desktop

Click 'Get Data' → 'Text/CSV'.

2

Select your CSV file

Navigate to the folder where you ran the script. Select the CSV file that was created.

3

Refresh when you want new data

Run sync_runner.py again to get fresh data. Then click Refresh in Power BI.
